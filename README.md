## API documentation

API documentation is available [here](http://docs.punnynames.apiary.io/#)

## Installation
```shell
# Clone repository:
git clone https://github.com/Dramloc/punny-names.git

# Go to directory:
cd punny-names

# Install node.js dependencies:
yarn

# Launch node server
npm start
```
